    class TavernaPlayer::Workflow

    This class represents a workflow. It is provided as an example model and can be extended or replaced as required.

    To replace the model completely please read about configuring the workflow model proxy in the “Taverna Player initializers” section of the ReadMe. To extend the model please read about “Overriding the default models and controllers” in the ReadMe.

    Public Instance Methods

    inputs → Hash click to toggle source

    Return a hash of information about this workflow’s inputs. The fields provided are:

    • :name - The name of the input port.

    • :description - A textual description (if provided) of the input port.

    • :example - Example input data (if provided) for the input port.

    All data is read directly from the workflow file itself.
