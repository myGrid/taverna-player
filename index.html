Synopsis

Taverna Player is a Ruby on Rails plugin to run Taverna workflows on a Taverna Server.

ZIP compatibility note

Version 0.8.0 and up of this gem uses version 1.0.0 and up of the rubyzip library. This has a backwards incompatible API that may clash with other gems that you are using. Please see the important note in the rubyzip readme for a workaround.
